### INSTALLATION

##### Requirements
This application is run with Python 3.7.5. You can download it [here][python_download_url].


### HOW TO USE

##### LAUNCH
Launch Legends of Runeterra and then run the following command in the terminal:
`
python app.py
`

- F2
